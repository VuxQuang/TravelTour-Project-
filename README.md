# Travel Tour Web Application

A full-stack Travel Tour web application built with **ReactJS** for the front-end, styled with **Tailwind CSS**, and powered by **Spring Boot (Java)** for the back-end. Users can explore, search, and book tours with a modern and responsive interface.

---

## Front-end Features
- Built with **ReactJS** for a dynamic and interactive user experience.
- Responsive UI using **Tailwind CSS**, optimized for both desktop and mobile.
- Display tour listings with images, descriptions, and pricing.
- Search and filter tours by location, price, or category.
- User authentication and profile management (login, register, update info).

## Back-end Features
- **Spring Boot** RESTful API to handle data operations.
- Tour management: CRUD operations for tours (create, read, update, delete).
- User authentication using **JWT (JSON Web Token)**.
- Database integration (**MySQL/PostgreSQL**) to store user and tour data.
- Secure endpoints for admin and user roles.

## Additional Highlights
- Full-stack project demonstrating **ReactJS + Tailwind CSS + Spring Boot** integration.
- Designed with clean code principles and modular architecture.
- Ready for deployment on cloud platforms like **Heroku** or **AWS**.

## Technologies Used
- **Front-end:** ReactJS, Tailwind CSS, Axios
- **Back-end:** Java, Spring Boot, Spring Security, JPA/Hibernate
- **Database:** MySQL or PostgreSQL
- **Authentication:** JWT (JSON Web Token)

## Project Status
- Fully functional with basic tour booking features.
- Can be extended with payment gateway integration and admin dashboard.
